# Adding a New City

## Getting Started

DetroGo supports addition of metro cities which allows it to expand with the help of the community anyone can add a new city but to add them you need to follow a certain format which DetroGo uses to read the city data even before that you’ll need to get the city’s metro network data from reliable sources the guide will go over which reliable sources you can look through for your cities data.


### Gathering the Data

You can get a cities metro data from quite a few sources offical and unoffical but the format may vary with the source which can be quite frustrating hence this guide will layout the exact steps to convert these formats to the one Detro accepts
